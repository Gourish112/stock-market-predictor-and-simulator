# 📈 Stock Market Predictor & Simulator  

**An AI-powered stock market prediction and simulation platform for data-driven trading decisions.**  

---

## 🚀 Project Overview  

The **Stock Market Predictor and Simulator** is a full-stack web application that utilizes AI/ML models to predict stock prices and provides a real-time trading simulator. The platform is designed to help users analyze stock trends, practice trading with virtual money, and optimize investment strategies.  

### 🔥 Key Features  

✅ **Stock Price Prediction** – AI-driven forecasts based on an **RNN model**.  
✅ **Real-time Stock Data** – Fetches live market data using **Yahoo Finance (`yf`)**.  
✅ **Paper Trading Simulator** – Buy/sell stocks using virtual money and track **profit/loss in real time**.  
✅ **Interactive Graphs** – Visualize real-time and predicted stock trends.  
✅ **Custom Prediction Intervals** – Choose between **1 day, 5 days, 1 month, or 1 year** forecasts.  
✅ **Transaction History** – Maintain a record of simulated trades.  

---

## 🛠️ Tech Stack  

- **Frontend:** React.js  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Machine Learning:** Python, TensorFlow/Keras (RNN model)  
- **API Integration:** Yahoo Finance (`yf`)  

---

## 📊 System Architecture  

- **Prediction Engine**: Uses an RNN model to forecast stock prices.  
- **Stock Data Fetching**: Real-time market data is retrieved via Yahoo Finance.  
- **Simulator**: Users can place buy/sell orders and track profits/losses.  
- **Frontend Dashboard**: Interactive UI for visualization and trading.

## 🛠️ Setup & Installation  

### 🔹 Backend Setup  
1. Clone the repository:  
   ```bash
   git clone https://github.com/Gourish112/stock-market-predictor-and-simulator.git
   cd your-repository
   ```
2. Install dependencies:  
   ```bash
   npm install
   ```
3. Start the backend server:  
   ```bash
   npm start
   ```

### 🔹 Frontend Setup  
1. Navigate to the frontend directory:  
   ```bash
   cd client
   ```
2. Install dependencies:  
   ```bash
   npm install
   ```
3. Start the frontend application:  
   ```bash
   npm start
   ```

## 📈 How to Use  

1. **Select a stock** to view real-time market data.  
2. **Choose a prediction interval** (1 day, 5 days, 1 month, or 1 year).  
3. **Analyze AI-generated predictions** and stock trends.  
4. **Use the simulator** to place buy/sell orders.  
5. **Monitor portfolio performance** and track historical transactions.
